1. 웹 브라우저(Chrome) - HTML문서를 해석하여 사용자 인터페이스 제공.

2. 태그(Tag), 요소(Element), 속성(Attribute)

3. HTML 페이지를 구성하는 부품 ==> 요소(Element) - Title, Body, Head, 이미지, 버튼, 라디오 버튼, 링크 주소...

4. 요소(Element)를 만들때 사용하는 작성 방법 ==> 태그(Tag)를 통해 요소(Element) 정의한다, 만든다

5. 자바스크립트에서는 요소(Element)를 객체(Object)라고 칭함.

6. 웹페이지는 두가지 방법 CSS 적용
    1. 내부 스타일 시트 적용 -- <style></style>
    2. 외부 스타일 시트 적용 -- <link>

7. 웹페이지는 두가지 방법 자바스크립트 적용
    1. 내부 자바스크립트 적용 -- <script></script>
    2. 외부 자바스크립트 적용 -- 외부에 js 파일 저장. 그리고 적용할 문서에 script src 속성을 이용하여 적용함.

8. 오류와 검증
    프로그램이 원하지 않는 방향으로 동작(버그) --> 버그를 잡는다(수정) = 디버그(debug)
    개발자도구 이용해서 오류와 검증을 한다. F12 or Ctrl + shift + i

9. HTML 기본 태그
    - 글자 태그
        * 앵커 <a></a>
        * 제목 <h1></h1>(1 ~ 7 까지 있음)
        * 글자 모양
    - 목록 태그
        * 순서가 있는 리스트
        * 순서가 없는 리스트
        * 중첩 리스트
    - 테이블 태그
        * 테이블과 셀 생성
        * 행 병합 / 열 병합
    - 미디어 태그(오디오, 비디오, 이미지)
        * 앵커
        * 글자 모양

10. 미디어 태그
    이미지, 오디오, 비디오(멀티미디어)를 HTML 문서에 삽입하는 태그
    <img> => src (이미지의 경로 지정), alt(이미지가 없을 때 나오는 글자 설정), width(이미지의 넓이), height(이미지의 높이)
    <audio></audio> => src (음악, 동영상 파일 경로 지정), preload, autoplay, loop, controls
    <video></video> => width, height
